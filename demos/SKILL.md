# demo rules

- Demos prove real consumer behavior; do not replace zero-code behavior with manual qyl calls.
- EFCore compiled model files are generated by EF tooling and may use System.Reflection; do not edit them by hand.
- If a demo needs an upstream library warning boundary, document it explicitly.
